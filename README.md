# 혼자왔니 Front

sever.js : node.js설치하고 저장한 경로로 가서 cmd에 

npm init -y

npm install express node-fetch

npm install axios

명령어를 입력하고

node server.js

를 입력해서 프록시 서버로 가동시켜야 웹 브라우저에서 CORS에러가 발생하지 않음
